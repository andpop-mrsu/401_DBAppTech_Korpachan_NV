# Guess Number

## Описание проекта
Программа для консольной игры "Угадай число".

### Правила игры

Компьютер загадывает число в диапазоне от 1 до максимального числа, указанного в настройках.
Игрок должен угадать число за несколько попыток (максимальное количество также задаётся в настройках).

После каждой попытки программа сообщает:
* если число игрока больше загаданного — выводит "меньше";
* если число игрока меньше загаданного — выводит "больше".

### Работа с базой данных

Игра автоматически сохраняет все партии в базу данных SQLite (`data/game.db`).

#### Сохраняемая информация

Информация о каждой игре сохраняется в базе данных SQLite:
* дата игры;
* имя игрока;
* максимально возможное число;
* загаданное число;
* исход игры (угадал/не угадал);
* количество попыток;
* список всех попыток (номер, введённое число, ответ компьютера).

#### Режимы работы программы
1. **Новая игра** - начать новую игру с возможностью указания параметров
2. **Список сохранённых игр** - просмотр всех сыгранных партий
3. **Список выигранных игр** - игры, в которых игрок победил
4. **Список проигранных игр** - игры, в которых игрок проиграл
5. **Статистика по игрокам** - количество побед и поражений, сортировка по победам
6. **Повтор партии** - просмотр всех ходов сохраненной игры

### Требования для запуска проекта
* PHP >= 7.4 (CLI-режим)
* Composer (глобально установленный)
* Расширение PHP: pdo_sqlite

### Установка и запуск

## Установка из GitHub

```bash
git clone --branch v1.2 https://github.com/Kornyshon/guess-number.git
cd guess-number
composer install
./bin/guess-number
```
## Глобальная установка из Packagist
```bash
composer global require Kornyshon/guess-number:1.2
guess-number
```
